# rabbitmq-test

This repository is made for a boilerplate that can be a milestone for the development process.  

## Build Normal
- ``` go build -v ./main.go ```
## Build with Docker
- ``` docker build -t rabbitmq-test .```


## Run Normal
- ``` go run main ```
## Run wtih Docker
- ``` docker run -d --name rabbitmq-test rabbitmq-test```
